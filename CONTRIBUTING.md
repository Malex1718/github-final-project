Contribuir al Proyecto de Calculadora de Interés Simple
Gracias por tu interés en contribuir a nuestro proyecto. Este documento proporciona las pautas y los pasos a seguir para contribuir efectivamente.
Cómo contribuir
Existen varias formas de contribuir a este proyecto:

Reportar errores (bugs): Si encuentras un error, por favor crea un issue en GitHub con una descripción detallada.
Sugerir mejoras: Si tienes ideas para mejorar el proyecto, puedes abrir un issue para discutirlas.
Enviar Pull Requests: Si deseas implementar una nueva función o arreglar un error, puedes enviar un Pull Request.

Proceso para Pull Requests

Bifurca (fork) el repositorio
Crea una nueva rama para tu característica o corrección:
Copiargit checkout -b feature/nombre-caracteristica
o
Copiargit checkout -b fix/nombre-correccion

Realiza tus cambios y haz commits con mensajes descriptivos:
Copiargit commit -m "Descripción detallada del cambio"

Envía tu rama a tu fork:
Copiargit push origin nombre-rama

Abre un Pull Request contra la rama principal del repositorio original

Estándares de código

Sigue las convenciones de estilo existentes en el proyecto
Incluye comentarios cuando sea necesario para explicar partes complejas del código
Mantén el script compatible con bash estándar para asegurar su funcionamiento en diferentes sistemas
Escribe pruebas para nuevas funcionalidades cuando sea posible

Guía para reportar issues
Al reportar un issue, incluye:

Una descripción clara del problema
Pasos para reproducir el error
El comportamiento esperado vs. el comportamiento actual
Capturas de pantalla si aplican
Entorno (sistema operativo, versión de bash, etc.)

Comunicación
Toda la comunicación debe ser respetuosa y seguir nuestro Código de Conducta.
Licencia
