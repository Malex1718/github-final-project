Proyecto de Calculadora de Interés Simple
Descripción
Este proyecto implementa una calculadora de interés simple que ayuda a los usuarios a calcular el interés generado en una inversión o préstamo basado en el capital principal, la tasa de interés y el tiempo.
Uso
Script de calculadora de interés simple
Copiarsimple-interest.sh principal tasa tiempo
Entrada:

principal: cantidad de dinero inicial (p.ej. 1000)
tasa: tasa de interés anual (p.ej. 5)
tiempo: período de tiempo en años (p.ej. 2)

Salida:

Interés simple calculado según la fórmula: capital * tasa * tiempo / 100
